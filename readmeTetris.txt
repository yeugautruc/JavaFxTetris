--------------------------------------------------------
Tetris version 1.0 from Group 26 - Programmieren WS2019
--------------------------------------------------------
________________________________________________________
Load GUI game with command:
	java -jar tetris.jar -gui
Load previous game and play in console with command:
	java -jar tetris.jar -load
Help information:
	java -jar tetris.jar -help
________________________________________________________
Button in GUI:
    play Button - play a new fresh tetris game 
    load Button - load previous saved game
    exit Button - exit game

    reset Button - reset GUI
    save Button - save gaming process in file
    highscore Button - show the highscore list
    mute/unmute Button - enable/disable background music
________________________________________________________
Action key:
	- w: rotate tetriminos
	- a: move tetriminos to  left
	- d: move tetriminos to right
	- s: move tetriminos 1 line down
	- x: move tetriminos all down
	- p: pause game (only for GUI)
	- +: increase game speed (only for GUI)
	- -: decrease game speed (only for GUI)
Notice: In console please type action key and hit enter.
________________________________________________________
Score:
	- each new Tetrisminos: 1 point
	- each full line: 10 points
________________________________________________________
HighScore:
	Highscore List record Top 10 Player with highest
	score. Please remember type your name after game
	finish to record your score in highscore list.
